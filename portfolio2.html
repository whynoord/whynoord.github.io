Overview

This application connects entrepreneurs with investors.

Features

Home: The home page displays all proposals except for the proposals owned by the currently logged-in user.

Detail: This page contains details of the proposal submission, including: Business name, submitting user, number of investors, number of donors, requested share, amount of money collected, etc.

Send Funds: On this page, the user can choose whether to invest or donate. If the user donates, the percentage of share obtained is 0%. If the user invests, the share obtained will be displayed depending on the investment amount.

Account: Contains a list of active proposals, businesses that have been invested in, and donated to.
